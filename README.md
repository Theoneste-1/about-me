# Portfolio with Next.js

## Overview

This is a personal portfolio website built with Next.js, a React framework for server-side rendering and static site generation. It showcases my projects, skills, and contact information in a modern, responsive design.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Project Showcase**: Displays a gallery of my work with descriptions and links.
- **Skills Section**: Highlights technical skills and tools I use.
- **Contact Form**: Allows visitors to send me messages directly.
- **Dark Mode**: Toggleable theme for better user experience.
- **SEO Optimization**: Meta tags and structured data for better search visibility.

## Technologies Used

- **Next.js**: For rendering and routing.
- **React**: Core library for components.
- **Tailwind CSS**: For styling and responsiveness.
- **Framer Motion**: For animations and transitions.
